### Prerequisites

- [Node.js 18](https://nodejs.org/en/)
- [Rust Stable](https://www.rust-lang.org/tools/install)

### Installation

Install yarn:

```bash
$ npm install yarn --global
```

Install dependencies:

```bash
$ yarn install
```
